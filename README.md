# 📚 StudyGO - Student Grade Overseer

StudyGO - Student Grade Overseer is an Android-based academic assistance application designed to help students monitor, evaluate, and improve their academic performance.

Students often struggle to understand how future quizzes, activities, and examinations affect their final grades. While traditional methods such as spreadsheets, calculators, or manual computation can track grades, they often fail to provide strategic insights for academic planning.

This application transforms grade tracking into a proactive academic decision-support system by allowing users to:

- Track their academic standing
- Analyze subject performance
- Simulate future score scenarios
- Identify assessments requiring more attention
- Explore realistic paths toward achieving target grades

The system utilizes a Backtracking Knapsack Algorithm to generate feasible score combinations that may help students reach their desired academic outcomes.

## 🧠 Core Algorithm

The system implements a Backtracking-based Knapsack Problem approach to:

- Explore possible score combinations
- Generate realistic academic paths
- Evaluate future assessment scenarios
- Determine whether a target grade is achievable

The algorithm recursively evaluates possible assessment outcomes while pruning impossible paths for efficiency.
